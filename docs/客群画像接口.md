# 群聊画像摘要接口（含用户画像与成员信息）

**接口名称**  
群聊画像摘要获取（含用户画像与成员列表）

**接口地址**  
`GET /api/groups/{roomid}/profile`

**用途**  
RPA 在触发回复前获取该群的历史概要 + 用户画像 + 成员信息。

**请求参数**
- `roomid`（path，必填）：上游企微群聊 roomid

**请求示例**
```http
GET /api/groups/wroCqZGwAAt4A4FqzQnLQqY3LtasIKfg/profile
```

**响应字段**
- `group_id`：群 ID  
- `group_name`：上游群名  
- `display_id`：显示名（优先映射值；无映射则上游群名）  
- `last_task_id`：最近一次分析任务 ID  
- `summary`：画像摘要  
  - `summary_text`：摘要文本  
  - `top_categories`：主要业务类别  
  - `top_themes`：核心问题主题（最多 5 条）  
- `topics`：详细主题列表  
  - `title` / `business_category` / `priority`  
  - `subtheme_count` / `closed_count`  
  - `top_questions`（最多 3 条）  
- `closed_stats`：闭环统计  
  - `total` / `closed` / `rate`  
- `members`：群成员画像列表（人工维护 + API 同步）  
  - `id` / `external_user_id`  
  - `name`  
  - `nickname`（群内昵称，人工）  
  - `real_name`（真实称呼，人工）  
  - `role`（内部/外部/其他）  
  - `profile_text`  
- `updated_at`：画像更新时间
