### Windows 环境（必须使用以下方法之一，否则中文乱码！）

**方法1：Python requests（强烈推荐，100%不乱码）**
```python
import requests
url = "http://diary.xiaosuantech.com:12010/api/reports/submit_by_token"
headers = {
    "Content-Type": "application/json; charset=utf-8",
    "X-API-Token": "b854618b7ad4fff2b0779fcd83512eb2"
}
data = {"content": "你的Markdown日记内容"}
response = requests.post(url, json=data, headers=headers)

```

**方法2：PowerShell（必须先设置编码）**
```powershell
# 步骤1：必须先执行这两行设置UTF-8编码！
chcp 65001 | Out-Null
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8

# 步骤2：准备数据并转为UTF-8字节
$content = @"
# 今日工作
- 工作内容...
"@
$body = @{content = $content} | ConvertTo-Json -Depth 10
$bytes = [System.Text.Encoding]::UTF8.GetBytes($body)

# 步骤3：发送请求
$headers = @{
    "Content-Type" = "application/json; charset=utf-8"
    "X-API-Token" = "b854618b7ad4fff2b0779fcd83512eb2"
}
Invoke-RestMethod -Uri "http://diary.xiaosuantech.com:12010/api/reports/submit_by_token" -Method Post -Body $bytes -Headers $headers
```

**方法3：curl.exe（注意是curl.exe不是curl）**
```powershell
# 先设置编码
chcp 65001 | Out-Null

# 使用curl.exe（Windows自带）
curl.exe -X POST "http://diary.xiaosuantech.com:12010/api/reports/submit_by_token" `
  -H "Content-Type: application/json; charset=utf-8" `
  -H "X-API-Token: b854618b7ad4fff2b0779fcd83512eb2" `
  --data-raw '{"content":"日记内容"}'
```

## 日记提交API
- 接口地址：http://diary.xiaosuantech.com:12010/api/reports/submit_by_token
- 请求方式：POST
- Headers：

- 请求体：

**重要提醒**：date 必须使用今天的日期（工具获取的当天日期），不可手写或推测！

- 接口地址：http://diary.xiaosuantech.com:12010/api/diaries/upload
- 请求方式：POST (multipart/form-data)
- Headers：X-API-Token: b854618b7ad4fff2b0779fcd83512eb2
- 表单字段：
  - file: 文件
  - date: 今天的日期

- 上传成功返回：{"success": true, "filename": "xxx.png", "url": "/api/diaries/attachments/..."}

## 在日记中引用图片
上传图片后，在Markdown中使用以下格式引用（使用返回的filename）：
```

```
系统会自动将文件名转换为完整URL显示图片。
